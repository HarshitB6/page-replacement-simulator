# PageSim — Page Replacement Algorithms Simulator

## Project Structure
```
project/
├── cpp/
│   ├── main.cpp       ← Entry point (C++ backend)
│   ├── fifo.cpp       ← FIFO algorithm
│   ├── lru.cpp        ← LRU algorithm
│   ├── optimal.cpp    ← Optimal algorithm
│   ├── lfu.cpp        ← LFU algorithm (NEW)
│   ├── utils.cpp      ← Helpers & analysis
│   ├── utils.hpp      ← Structs & declarations
│   └── json.hpp       ← nlohmann/json (header-only)
├── web/
│   ├── index.html     ← Main UI
│   ├── style.css      ← Styles (dark/light mode)
│   └── script.js      ← JS simulation + visualization
├── data/
│   └── output.json    ← Generated by C++ backend
└── README.md
```

## How to Run

### Option A — Browser Only (no compilation needed)
1. Open `web/index.html` in a browser
2. Enter page reference string and frames
3. Click **Run Simulation**

The JS engine implements all 4 algorithms natively — no backend required.

### Option B — With C++ Backend
```bash
# Compile (from cpp/ directory)
cd cpp
g++ -std=c++17 -O2 main.cpp fifo.cpp lru.cpp optimal.cpp lfu.cpp utils.cpp -o ../run

# Run interactively
../run

# Or pass args directly:
../run 3 1 2 3 4 1 2 5 1 2 3 4 5
#       ^ frames  ^ page sequence
```

This writes `data/output.json` which the browser auto-loads on next open.
